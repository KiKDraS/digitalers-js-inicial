Conceptos básicos

¿Qué la programación?

La programación es la habilidad de escribir, organizar, y ejecutar instrucciones para que un computador realice tareas específicas.

Todo proceso de Programación arranca entendiendo cuál es mi objetivo final

Programar NO es escribir un Lenguaje de Programación

Programar es organizar y ejecutar las acciones necesarias para lograr el objetivo final

¿Qué es un Programa?

Un programa es un conjunto de instrucciones que se agrupan con la intención de lograr el objetivo final

Lenguaje de Programación

Un Lenguaje de Programación es una herramienta que permite comunicarse con el elemento encargado de ejecutar el programa

¿Cómo elijo Lenguaje de Programación?

  • ¿Qué problema tengo que resolver?
  • ¿Quién tiene que resolver el problema?
  • ¿Qué limitantes tiene el Lenguaje?
  • ¿Qué tan cómo me siento con el Lenguaje?

Tipos de Lenguaje

Todo Lenguaje de Programación tiene tipo de datos (un tipo de dato permite otorgar información)

  • Forma de tipado - Qué tan estricto es el Lenguaje con el Programador para forzarlo a escribir el tipo de dato
    • Fuertemente tipado - El Programador está obligado a escribir siempre el tipo de dato. Java y C#
    • Debilmente tipado - El Programador puede no escribir el tipo de dato porque el elemento encargado de ejecutar el programa tiene la capacidad en inferir el tipo según el valor. Javascript y Kotlin
  • El tipo de tipado - El Lenguaje permite que el tipo de dato cambie a lo largo de la ejecución del programa
    • Tipado estático - No se permite el cambio del tipo de dato. Java, Kotlin
    • Tipado dinámico - Se permite el cambio del tipo de dato. Javascript, python

Lenguaje Javascript

Lenguaje pensado para el trabajo con los navegadores web. Por este motivo se lo conoce como "Lenguaje web". Fue creado en el año 1995 por Brendan Eich

Evolución del Estandar Lenguaje

El Estandar del Lenguaje son la reglas sintáctico gramaticales que permiten la escritura del mismo

  • 1995 - creado por Brendan Eich
    1. ECMAScript 1st Edition - 1997
    2. ECMAScript 2nd Edition - 1998
    3. ECMAScript 3rd Edition - 1999
    4. ECMAScript 4th Edition - 2000 (no está disponible)
    5. ECMAScript 5th Edition - 2009 (Estandar que todo navegador reconoce en su totalidad)
      • var - Única palabra reservada para que el Lenguaje sepa que queremos crear una variable
    6. ECMAScript 6th Edition (ES6) - 2015
      • var - Deja de ser recomendado su uso
      • let - Palabra reservada para que el Lenguaje sepa que queremos crear una variable
      • const - Palabra reservada para que el Lenguaje sepa que queremos crear una variable
    7. ECMAScript 7th Edition (ES7) - 2016
    8. ECMAScript 8th Edition (ES8) - 2017
    9. ECMAScript 9th Edition
  • 2009 - Ryan Dhal crea Node.js
    • Node.js es un interprete de Javascript que no requiere de un navegador
